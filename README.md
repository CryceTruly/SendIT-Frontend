## What is SendIT? :open_mouth:

- **Too Cool** - SendIt is a courrier service app that enables customers to send items to different places

#### Contributing :smiley:
```
- clone the project
- npm install
- npm run start
```
#### Build for production

```
npm run build
```
#### Run Tests
`
npm test
`

### Run tests with coverage

` npm test -- --coverage
`
